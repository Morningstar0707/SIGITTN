# Sistema de Gestión de Infraestructura TTN (SIGITTN)

Aplicación Web Progresiva (PWA) diseñada para el **Terminal de Transporte de Neiva**, que permite a los funcionarios registrar, gestionar y hacer seguimiento a **tickets de daños o fallas** en la infraestructura física y tecnológica.

---

## 📋 Descripción del Proyecto

El **Sistema de Gestión de Infraestructura TTN (SIGITTN)** es una plataforma que facilita la administración y control de incidencias relacionadas con la infraestructura del terminal.

La aplicación permite a los funcionarios:

- Crear **tickets** para registrar daños o fallas.
- Clasificar los tickets según su tipo.
- Realizar seguimiento al estado de cada ticket.
- Consultar el historial de incidencias registradas.
- Gestionar el mantenimiento de la infraestructura del terminal.

Este sistema busca optimizar los procesos internos de registro y seguimiento de incidencias, permitiendo una atención más organizada y eficiente de los problemas de infraestructura.

---

## 🎯 Objetivo

Desarrollar una **Aplicación Web Progresiva (PWA)** que permita centralizar la creación y gestión de **tickets de mantenimiento e infraestructura** en el Terminal de Transporte de Neiva, facilitando el control y seguimiento de incidencias por parte de los funcionarios.

---

## 👥 Usuarios del Sistema

El sistema está dirigido a aproximadamente **100 funcionarios del Terminal de Transporte de Neiva**, quienes podrán:

- Registrar tickets de daños o fallas.
- Consultar el estado de los tickets.
- Hacer seguimiento a las incidencias reportadas.

---

## ⚙️ Funcionalidades Principales

- Creación de **tickets de incidencias**.
- Clasificación de tickets (infraestructura física o tecnológica).
- Seguimiento del estado del ticket.
- Consulta del historial de tickets.
- Gestión de tickets por parte de los funcionarios responsables.

---

## 🏗️ Tipo de Aplicación

El sistema se desarrollará como una:

**Aplicación Web Progresiva (PWA)**

Las PWA permiten:

- Acceso desde dispositivos móviles y computadores.
- Instalación como aplicación en el dispositivo.
- Mejor experiencia de usuario.
- Uso eficiente incluso con conexiones limitadas.

---

## 📂 Alcance del Proyecto

El sistema permitirá gestionar tickets relacionados con:

- Equipos tecnológicos
- Redes y conectividad
- Infraestructura física
- Equipamiento del terminal
